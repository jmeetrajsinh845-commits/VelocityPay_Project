# VelocityPay Analytics Platform

## Project Overview

VelocityPay Analytics Platform is an end-to-end Data Analytics and Machine Learning project built for a fictional FinTech company. The project demonstrates how raw operational data can be transformed into business insights, predictive analytics, and executive dashboards.

The objective of the project is to simulate a real-world Data Analyst / Data Analytics workflow, starting from data generation and exploratory analysis to customer segmentation, churn prediction, and interactive Power BI reporting.

---

# Business Problem

VelocityPay provides digital payment solutions, subscription-based services, SME lending, and customer engagement products.

The company faces several business challenges:

- Increasing customer churn
- Failed payment transactions
- Revenue leakage
- Loan portfolio monitoring
- Customer segmentation
- Product usage analysis
- Executive business reporting

The purpose of this project is to identify these problems using analytics and provide actionable business insights.

---

# Project Objectives

The project focuses on the following objectives:

- Analyze customer transactions
- Measure payment success rate
- Detect revenue leakage
- Analyze customer churn
- Identify churn reasons
- Measure churn impact on revenue
- Perform customer segmentation using RFM Analysis
- Build a Machine Learning model for churn prediction
- Explain important prediction factors
- Build interactive Power BI dashboards
